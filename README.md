# Marta's Mirror

Live app: [https://davbachman.github.io/MartasMirror/](https://davbachman.github.io/MartasMirror/)

## What It Does

Marta's Mirror is a browser-based image inversion/mirror app with drawing overlays and PNG export.

## Instructions for Use

1. Open the live app link above.
2. Click **Upload Image** to load a JPG/PNG/WebP image.
3. Use **Flip H** and **Flip V** to mirror the image across the horizontal or vertical axis.
4. Use the **Rotation** slider to rotate the image.
5. Use **Zoom** to change the inversion circle size and **Image Scale** to resize the uploaded image.
6. Use **Move** mode to:
   - drag the image inside the inversion circle
   - drag the inversion circle
   - select drawn shapes
   - edit selected shapes (drag line endpoints, move/resize circles)
7. Use **Line** or **Circle** mode to draw green overlays on the canvas.
8. Click a drawn shape in **Move** mode, then press **Delete** or **Backspace** to remove it.
9. Use **Undo** and **Clear** for overlay history management.
10. Click **Export PNG** to save the current canvas (image + overlays) as a bitmap.
