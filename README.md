# Marta's Mirror

Live app: [https://davbachman.github.io/MartasMirrorV2/](https://davbachman.github.io/MartasMirrorV2/)

## What It Does

Marta's Mirror is a browser-based image inversion and stereographic mirror app with full-plane multi-image composition, drawing overlays, project save/import, and PNG export.

## Instructions for Use

1. Open the live app link above.
2. Click **Upload Images** to load one or more JPG/PNG/WebP images.
3. Click an image thumbnail or click an image on the canvas to select or deselect it.
4. Use **Flip H** and **Flip V** to mirror the selected image across the horizontal or vertical axis.
5. Use the **Rotation** slider to rotate the selected image.
6. Use **Zoom** to change the inversion circle size and **Image Scale** to resize the selected image.
7. Use **Move** mode to:
   - drag the selected image anywhere on the canvas
   - drag the inversion circle
   - drag the whole scene when no image is selected
   - select drawn shapes
   - edit selected shapes (drag line endpoints, move/resize circles)
8. Use **Line** or **Circle** mode to draw green overlays on the canvas.
9. Click a drawn shape in **Move** mode, then press **Delete** or **Backspace** to remove it.
10. Use **Undo** and **Clear** for overlay history management.
11. Click **Save** to download a self-contained project file with embedded image assets.
12. Click **Import** to restore a saved project.
13. Click **Export PNG** to save the current canvas (image + overlays) as a bitmap.
